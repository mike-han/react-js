- Copy [`arcgis-charts-js-v1.0.0-beta.208`](https://devtopia.esri.com/WebGIS/arcgis-charts/releases/tag/arcgis-charts-js-v1.0.0-beta.208) into `src/arcgis-charts/v1.0.0-beta.208`

- Run `npm start`
